2048

The game 2048 is a popular game on the App Store. The goal of the game is to combine two tiles of the same number to reach the next power of 2. For example, combining two tiles with the number 2 results in one tile with the number 4. The objective of the game is to get a tile with the largest number of 2 on it possible. After each move, a new tile is added by the computer to the board in a random location.


The AI plays this game using the Expectimax algorithm. The computer creates a tree that stores all possible moves, then all possible moves for that move, all possible moves for each of those moves, and so on and so forth until all possible options reach the end-of-game state, which contains a reward based off of the game score.


To decide what move to make next, the computer uses this tree to pick the best move based off of the reward at the end and using random moves to decide where the computer will generate the next tile.
